# ☠️ SUB-SUCKER ZERO

**A high-performance DOM-Scraping Extraction Engine for YouTube Subscriptions.**

Standard YouTube APIs severely limit how you can interact with your own subscription data. **Sub-Sucker Zero** is a Manifest V3 browser extension built to bypass standard API restrictions. It aggressively scrapes the DOM to extract, filter, and export your entire subscription list into a clean, localized PDF report.

---

## Core Systems

| Module | Function |
|---|---|
| **The Abyss Diver** | Auto-scrolling script that bypasses lazy-loading to force-render the entire list. |
| **Visual Telemetry** | Extracts and resolves encrypted avatar image URLs directly from the DOM. |
| **Target Locator** | Real-time, high-contrast search matrix to instantly filter targets. |
| **The PDF Engine** | Compiles the raw HTML data into a clean, printable intelligence report. |

## ⚠️ Installation Protocol (Developer Override)

*Notice: Because this tool utilizes aggressive scraping algorithms, it is classified as "Unauthorized" by the Chrome Web Store. You must install this using Developer Override Mode.*

1. Download the `Sub-Sucker-Zero_v1.0_DevBuild.zip` file from this repository.
2. Extract the `.zip` file to a folder on your computer.
3. Open your Chromium browser (Chrome, Brave, Edge) and navigate to `chrome://extensions/`.
4. Toggle **Developer mode** `[ON]` in the top right corner.
5. Click **Load unpacked** in the top left and select the extracted folder.
6. Pin the extension to your toolbar to initialize the extraction engine.
